# hyperledger-fabric-external-chaincode-server

## Requirement
- golang 1.14
- docker
- docker-compose

## Usage
1. To launch the chain
```bash
$ make start
``` 

2. To call the chaincode function
```bash
$ make initChainCode
$ make invokeChainCode
$ make queryChainCode
```
